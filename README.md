### Brock University WiFi Occupancy Tracker

A tool to monitor floor Occupancy, display stats and data collection.

To deploy locally, just clone the repository and run -
```./gradlew clean build``` and then
```./gradlew run```

The standalone Python script is available in ```app/src/main/python``` directory.

#### System Requirements

Java 21 or above needs to be installed, and JAVA_HOME needs to added to the file path.
